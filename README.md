# OxEdgeHack
Oxford Edge Responsible AI Hackathon Nov 01 2025


## Installation
1. Clone the repository
2. Create python environment (optional)
3. Install `requirements.txt`
4. Run `pip install gradio`

## Usage
1. Export the anthropic API key `export ANTHROPIC_API_KEY=<key>`
2. Store documents in the `/docs/` directory
3. Use the command `python3 src/backend/rag_bot.py`
4. Navigate to `localhost:7860` or `http://127.0.0.1:7860/` to use the bot
